## Utilizzo

1. Assicurarsi di avere Python 3.10+ installato.
2. Installare i requisiti:
   ```bash
   pip install -r requirements.txt
   ```
3. Rendere eseguibile lo script:
   ```bash
   chmod +x run_all_tests.sh
   ```
4. Eseguire tutti i test:
   ```bash
   ./run_all_tests.sh
   ```
